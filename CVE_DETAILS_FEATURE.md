# Фича: Отображение деталей CVE

## Описание
При нажатии на пункт меню «Детали» нужно показывать подробную информацию о выбранном CVE.

## Логика блоков

### Блок 1: Модальное окно (Dialog) для просмотра деталей
- **Тип**: Dialog (модальное окно)
- **Триггер**: Нажатие на пункт меню «Детали»
- **Контент**:
  - CVE ID
  - Severity (с StatusBadge)
  - Статус (с StatusBadge)
  - CVSS Score (с цветовой индикацией)
  - Published Date
  - Описание
  - Кнопка "Закрыть"

### Блок 2: Связь с контекстом
- При нажатии на «Детали» в меню нужно передавать `selectedCVE` в Dialog
- Dialog должен открываться при нажатии на «Детали»
- Dialog должен закрываться по кнопке "Закрыть"

## Дизайн-система
- Использовать существующие компоненты: Dialog, StatusBadge, Button
- Использовать дизайн-токены: `--surface-card`, `text-color-secondary`, `text-color-danger`, `text-color-warning`, `text-color-success`
- Следовать типографике и отступам дизайн-системы

## Интеграция с существующим кодом
- Использовать существующее состояние `showDetails` и `selectedCVE`
- При нажатии на «Детали» установить `selectedCVE` и `showDetails = true`
- Dialog должен быть стилизован в соответствии с дизайн-системой
