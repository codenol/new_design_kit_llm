# Фича: Bulk Actions для таблицы CVE

## Описание

Добавить возможность массового выполнения действий над выбранными строками таблицы CVE.

## Логика работы

### 1. Выделение строк
- Использовать существующий checkbox в колонке selectionMode="single" → изменить на "multiple"
- При клике на строку выделяется одна или несколько строк

### 2. Панель bulk actions
- При выделении 2+ строк показывать кнопку "Bulk Actions" над таблицей
- Кнопка скрыта когда выделена 0 или 1 строка

### 3. Доступные действия
- **Отправить на исправление** — отправить все выбранные CVE на исправление
- **Архивировать** — переместить выбранные CVE в архив
- **Сбросить выделение** — снять выделение со всех строк

### 4. Визуальный стиль
- Использовать дизайн-систему проекта (цвета, шрифты, отступы)
- Кнопки bulk actions в стиле outlined
- Подтверждение действий через toast-уведомления

## UI Компоненты

```
[Фильтры и поиск] [Bulk Actions] [Сбросить выделение]
```
