# Страница "Таблицы PostgreSQL"

## Описание
Страница отображает все таблицы базы данных PostgreSQL с их полями, типами данных и описаниями.

## Структура страницы

### 1. Шапка (Header)
- Логотип и название приложения
- Навигационное меню (сайдбар)
- Кнопка "Обновить" для перезагрузки списка таблиц

### 2. Основной контент (Main Content)
- Заголовок страницы: "Таблицы PostgreSQL"
- Кнопка "Добавить таблицу" (если поддерживается)
- Кнопка "Экспорт в CSV"

### 3. Таблица данных
Столбцы таблицы:
- **Название таблицы** - имя таблицы (например, `users`, `roles`, `permissions`)
- **Количество строк** - количество записей в таблице
- **Поля таблицы** - список полей с типами данных
- **Описание** - описание таблицы (если есть)
- **Действия** - кнопки для просмотра структуры, редактирования, удаления

### 4. Модальное окно "Структура таблицы"
При клике на таблицу открывается модальное окно с подробной структурой:
- Название таблицы
- Список всех полей с:
  - Именем поля
  - Типом данных
  - Размером (если применимо)
  - Nullable/Not null
  - Default значение
  - Описание

## Дизайн-система
- Использовать компоненты из `design-system/uikit`
- Цвета из `design-system/tokens/_theme.scss`
- Типографика из `design-system/layout/_typography.scss`
- Сетка из `design-system/layout/_content.scss`

## Функциональность
- Отображение списка всех таблиц PostgreSQL
- Фильтрация по названию таблицы
- Поиск по полям
- Сортировка по колонкам
- Экспорт данных в CSV
- Валидация форм (если есть форма добавления)
